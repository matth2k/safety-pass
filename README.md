# Safety Pass: Compiler Passes and Pattern Rewriting for Circuits

## Description

A Rust library for orchestrating compiler pass pipelines on safety-net netlists.

You can read the docs for the library [here](https://matth2k.github.io/safety-pass/), or you can install the compiler driver tool with `cargo install nl_opt`. Examples for how to use the [library](https://crates.io/crates/safety-pass) and [Verilog compiler](https://crates.io/crates/nl_opt) can be found on their respective crates.io link.
